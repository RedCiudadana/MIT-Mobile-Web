Credits

The Mobile Web was developed using the Mobile Web Open Source Project (OSP). The Mobile Web OSP is based on version 0.9 of MIT Mobile Web. Mobile Web OSP was modified & documented by Dave Olsen of West Virginia University's University Relations/Web Services.

The original MIT Mobile Web was developed by MIT's Information Services & Technology (IS&T) as a service to the MIT community. The initial design and implementation took place in January-May 2008, and after a beta release and testing in May 2008, the site was officially launched in June 2008.

  • Project leader: Andrew Yu (Mobile Devices Platform Coordinator, IS&T)
  • Development and integration: Brian Patt, Sonya Huang, and Albert Chow
  • User interface design: Eric Kim
  • Team members: Matthew Sullivan, Justin Anderson, and Joanna Proulx
  • Advisor: Taeminn Song
